Overview

This project is a menu-driven Employee Management System built using C#.
It demonstrates key programming concepts like OOP, Inheritance, Polymorphism, Collections, LINQ, Exception Handling, and Database Connectivity (ADO.NET).

🚀 Features
👨‍💼 Employee Management

Add Full-Time Employees

Add Contract Employees

View all employees with calculated salary

Delete employee by ID

🔍 Search & Sort

Search employees by department

Sort employees by salary (High → Low)

📊 Bonus Features

Display highest paid employee

Calculate average salary

Group employees by department

🗄️ Database Integration

Connects to SQL Server using ADO.NET

Fetches employee data from database table

🧠 Concepts Used

Object-Oriented Programming (OOP)

Inheritance & Method Overriding

Polymorphism

Collections (List<T>)

LINQ (Where, OrderBy, GroupBy, Average)

Exception Handling

ADO.NET (SQL Connection, Command, DataReader)
How to Run

Open project in Visual Studio

Build the solution

Run the program

Use menu options to perform operations
Learning Outcomes

Understanding of real-world OOP design

Hands-on experience with LINQ

Database connectivity using C#

Building menu-driven applications
